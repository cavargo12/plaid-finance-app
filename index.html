Plaid to Sheets Personal Finance Automation

This is a backend application developed for personal use. It leverages the Plaid API (link.plaid.com) to securely retrieve financial transaction and investment data from my connected financial institutions.

The retrieved data is automatically organized and stored in a private Google Sheet for personal budgeting, analysis, and financial tracking.

This application runs as a Google Cloud Function and does not have a public user interface. Its primary goal is to automate personal financial data aggregation for improved personal financial management.
